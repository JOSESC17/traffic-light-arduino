# traffic-light-arduino
Traffic control system implementation: transitioning from automatic loops to interactive push-button triggers.

###Arduino Traffic Light Control 🚦 This repository documents my first mechatronics engineering project, where I explore peripheral control and embedded programming logic.

###Project Stages

Automatic Cycle (v1) An open-loop system that alternates lights (Green, Yellow, Red) using fixed timers. This stage is designed to validate hardware connections and basic timing.

Interactive Cycle (v2) A push-button is integrated to manually control state transitions. The system responds to an external input, replacing fixed timing with user-triggered reactive logic.

###Requirements Arduino Uno.

3 LEDs and 220-ohm resistors.

1 Push-button.

Breadboard and jumper wires.
